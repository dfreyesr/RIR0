<h3>Descripción Historias de Usuario</h3>

|Elemento|Descripción|
|----------|----------|
|Personalización de Equipos y Rutinas|**Como** entrenador de un grupo de deportistas, **Quiero** tener la capacidad de seleccionar deportistas específicos desde la aplicación web, **Para** poder crear un equipo personalizado y asignarles rutinas de entrenamiento adaptadas a sus necesidades individuales|
|Criterios de aceptación|<ul><li>Debe haber una opción en la interfaz para que el entrenador seleccione deportistas individuales y los agregue a un equipo.</li><li>Al seleccionar a un deportista, el entrenador debe poder acceder a una lista de rutinas predefinidas y personalizadas.</li><li>Después de seleccionar una rutina, el sistema debe permitir al entrenador asignarla al equipo o a un deportista específico, asegurando que cada deportista tenga una rutina adecuada.</li></ul>|
